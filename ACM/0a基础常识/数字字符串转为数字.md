istringstream和ostringstream

头文件

`#include<sstream>`

```cpp
template<typename T> string toString(const T& t){
    ostringstream oss;  //创建一个格式化输出流
    oss<<t;             //把值传递如流中
    return oss.str();  
}
```

```cpp
Type stringToNum(const string& str){ 
    istringstream iss(str); 
    Type num; 
    iss >> num; 
    return num;     
} 
```



`atoi(const char*)`

- 参数是c的字符串，如使用string需先调用`c_str()` 方法
- 不会检查范围，如果越界则会返回上下界



`stoi(const string*,from,base)`

- 如果越界会报错

- 将base进制的string转化为十进制

`string.c_str()`

将string转为const char*





`to_string(int x)`

将数值x转为字符串
