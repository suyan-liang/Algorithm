

#### 素数

判断一个数是否为素数

```cpp
bool isprime(int x) {
	if (x < 2) return false;
	if (x == 2) return true;
	for (int i = 2; i * i <= x; i++) { //注意这里是<=
		if (x % i == 0) return false;
	}
	return true;
}
```





#### 素数筛

给出一个布尔数组，其下标代表数字，值代表该数是否为素数，是为1，否为0



素数筛的思想

`A[0..N]`

首先标记所有的数为素数,标记0和1为非素数

***从2开始循环到n***

***如果当前数是素数，则标记其所有倍数为非素数***

```cpp
bool isprime[100+1];

for (int i = 0; i <= 100; i++) {
		isprime[i] = true;
	}
	isprime[0] = isprime[1] = false;
	for (int i = 2; i <= 100; i++) {
		if (isprime[i]) {
			for (int j = 2 * i; j <= 100; j += i) {
				isprime[j] = false;
			}
		}
	}
```
